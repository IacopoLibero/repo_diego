Introduzione a Bootstrap

Bootstrap è un framework CSS che facilita la creazione di siti web responsive e moderni. Utilizza un sistema a griglia flessibile che permette di creare layout che si adattano a diverse dimensioni di schermo.

Design Responsive

Il design responsive è una tecnica che permette al layout di un sito web di adattarsi automaticamente alla dimensione dello schermo del dispositivo su cui viene visualizzato. Bootstrap utilizza classi CSS per gestire il design responsive.

breakpoint

Bootstrap definisce diversi "breakpoint" per i diversi tipi di dispositivi. I breakpoint sono i punti in cui il layout del sito web cambia per adattarsi alle dimensioni dello schermo. I breakpoint predefiniti di Bootstrap sono:

  • Extra small (): schermi più piccoli di 576px
  • Small (sm): 575px <= grandezza < 768px
  • Medium (md): 768px <= grandezza < 992px
  • Large (lg): 992px <= grandezza < 1200px
  • Extra Large (xl): grandezza >= 1200px

per ogni dubbio consulta la documentazione
