Cos'è HTML?

HTML (HyperText Markup Language) è il linguaggio standard per creare pagine web.
Utilizza una serie di elementi e tag per strutturare il contenuto di una pagina.

Struttura di base di un documento HTML

                
    <!DOCTYPE html>
    <html>
    <head>
        <title>Titolo della pagina che vedi sulla scheda</title>
    </head>
    <body>
        <header>
            Questa è l'intestazione della pagina.
        </header>
        <main>
            Questo è il contenuto principale della pagina.
        </main>
        <footer>
            Questo è il piè di pagina della pagina.
        </footer>
    </body>
    </html>

Elementi principali

  • <h1> - <h6>: Intestazioni
  • <p>: Paragrafi
  • <a>: Link
  • <img>: Immagini
  • <ul> e <ol>: Liste non ordinate e ordinate
  • <div> e <span>: Contenitori generici
  • <table>: Tabelle
